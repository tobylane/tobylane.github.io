#define luaT_new (   L,
  T 
)    new ((T*)lua_newuserdata(L, sizeof(T))) T

Version of operator new which allocates into a Lua userdata.

If a specific constructor of T is required, then call like: T* variable = luaT_new(L, T)(constructor arguments); If the default constructor is wanted, it can be called like: T* variable = luaT_new(L, T); See also luaT_stdnew() which allocates, and also sets up the environment table and metatable for the userdata.
