THDrawable * THMap::hitTest ( int  iTestX,
int  iTestY 
) const

Perform a hit-test against the animations attached to the map.

If there is an animation at world pixel co-ordinates (iTestX, iTestY), then it is returned. Otherwise nullptr is returned. To perform a hit-test using world (tile) co-ordinates, get the node itself and query the top 8 bits of THMapNode::iFlags, or traverse the node's animation lists.
